
1) Считать csv файл с помощью функций для работы с csv файлами, такими как fgetcsv
2) Работа с БД будет отличаться в зависимости от используемого инструментария (нативного PHP или
использования фреймверков).
3) В целом для начала необходимо проверить существуют ли данные записи в БД. Проверка будет осуществляться
по ID товара или любому другому уникальному полю. Если записи в БД есть, их необходимо обновить,
если нет - добавить. Если говорить о конкретных функциях, для работы с БД mySql можно воспользоваться
следующими: mysqli_connect, mysqli_execute и т.д. Эти функции являются псевдонимами статических функций
класса mysqli. Можно воспользоваться его методами и работать через этот класс.
В ситуации когда используються фреймверки (например Symfony) можно воспользоваться его функционалом.
Entity, Mapping, Migration и т.д. Сам механизм основан на Doctrine (это удобный функционал для работы с БД с помощью PHP).
4) Перед добавлением или изменением данных в БД необходимо проверить данные которые нужно будет добавить
или изменить. Воспользовавшись например функцией нативного PHP htmlspecialchars. Так же необходимо
использовать плэйсхолдеры в запросах.
